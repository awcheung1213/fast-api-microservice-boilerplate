# fast-api-microservice-boilerplate
Contains an out of the box fast-api microservice

## Description
An out-of-the-box microservice leveraging the following languages, frameworks, and technologies:
- fast-api
- Terraform
- SQLAlchemy
- Datadog
- Google Cloud Platform
  - CloudSQL
  - Redis
  - Secrets Manager
